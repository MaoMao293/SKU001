# Sound Lab Setup Package

This repository contains all necessary files to set up and use the Sound Level Meter software.

## How to Download

Please refer to `Download.png` for step-by-step instructions on how to download the required files.

## Files Included

| File | Description |
|------|-------------|
| `Download.png` | **Tutorial screenshot** - shows how to download the installation package and manual |
| `README.md` | This file - package overview and instructions |
| `Sound Level Meter Instruction Manual.pdf` | Complete user manual and operating instructions for the sound level meter |
| `SoundLabSetup.zip` | Installation package - contains all required setup files to install the software |

## Quick Start

1. **Download** - Follow the steps shown in `Download.png` to download the files
2. **Read** - Open `Sound Level Meter Instruction Manual.pdf` for detailed operating instructions
3. **Install** - Extract and run setup files from `SoundLabSetup.zip`

## Installation

To install the Sound Level Meter software:

1. Download `SoundLabSetup.zip` (see `Download.png` for guidance)
2. Extract the ZIP file to your desired location
3. Run the setup executable to complete installation
4. Refer to the Instruction Manual for usage guidelines
